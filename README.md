# 更多提醒铃声 (SillyTavern Chime)

## 简介

这是一个为SillyTavern聊天应用设计的扩展，用于在收到新消息时播放提示音效，增强用户体验。

## 功能

- 在收到新消息时播放自定义提示音
- 支持多种音频包选择
- 简单易用的设置界面
- 可随时启用/禁用提醒功能

## 使用说明

1. 在SillyTavern的扩展设置中找到「更多提醒铃声」
2. 勾选「启用插件」开关
3. 从下拉菜单中选择喜欢的音频包
4. 点击「测试消息音效」按钮可以预览音效

## 音频包

目前提供以下音频包：

- **default** - 标准提示音
- **doubao** - doubao风格提示音
